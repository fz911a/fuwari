# 刷不死uboot 
### 网线传输
```bash
# ttl连接 频率11520
setenv serverip 192.168.1.10
setenv ipaddr 192.168.1.1
setenv loadaddr 0x89000000
tftp -i 192.168.0.1 put  刷文件名字.itb
```
### ttl传输（使用Tera term）
```bash
# ttl连接 频率11520
loady 0x89000000    #文件-传输-YMODEM-发送 
```

#### 群内版
```bash
setenv one flash read 0x600000 0x100000 \$loadaddr
setenv two "; bootm"
setenv bootcmd "$one$two"
saveenv
flash erase 0x600000 0x100000
flash write 0x600000 0x100000 0x89000000
reset
````
#### gh版
```bash
flash erase 0x600000 0x100000
flash write 0x600000 0x100000 0x89000000
setenv one flash read 0x602100 0x0FDF00 \$loadaddr
setenv two "; bootm"
setenv bootcmd "$one$two"
saveenv
reset
```
# 其他命令

```bash
/etc/init.d/network restart  # 重启网络
/etc/init.d/uhttpd restart   # 重启web界面
apk add --allow-untrusted /tmp/upload.apk   # 强制安装
apk upgrade --allow-untrusted packgename --force-missing-repositories   # 强制更新
# 清理 LuCI 网页后台的旧缓存（解决菜单错乱） 
rm -rf /tmp/luci-* 
# 清理 apk 包管理器的下载缓存（释放存储空间） 
rm -rf /var/cache/apk/*
```
